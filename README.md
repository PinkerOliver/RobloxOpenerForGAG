# GAG Shop Opener

A lightweight WinForms utility that automatically brings Roblox to the foreground every 5 minutes.

## Features

- Runs in the background
- Automatically detects the Roblox client
- Restores the Roblox window if minimized
- Brings Roblox to the foreground every 5-minute interval

## Requirements

- Windows
- .NET Framework / .NET
- Roblox desktop client

## Usage

1. Launch Roblox.
2. Start GAG Shop Opener.
3. Leave it running.

##IT DOESN'T GO AGAINST ROBLOX TOS
